Block types

The editor itself is made up of Blocks. The default ones are:

  • text Text
  • image Image
  • quote Quote
  • list List
  • twitter Tweet
  • add Many more

There are more blocks available in the dedicated blocks repository or it's incredibly easy to write your own.

For more information on Sir Trevor JS check out the repository at github.com/madebymany/sir-trevor-js
